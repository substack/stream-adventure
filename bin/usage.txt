Usos:

  stream-adventure

    Mustra el menu interactivo para seleccionar la aventura que quieras hacer.
 
  stream-adventure list 

    Muestra una lista de todas las aventuras.

  stream-adventure select NAME 

    Selecciona una aventura.

  stream-adventure current

   Muestra la aventura actualemente seleccionada.

  stream-adventure run program.js

   Corre tu programa con el correspondiente input.

  stream-adventure verify program.js

    Verifica tu programa ante la salida esperada.
